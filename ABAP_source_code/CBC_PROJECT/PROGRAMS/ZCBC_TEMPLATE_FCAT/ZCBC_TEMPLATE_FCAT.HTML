PERFORM SET_FCAT.

FORM EDIT_FIELDCATALOG USING PV_FIELDNAME TYPE LVC_S_FCAT-FIELDNAME
                                 PV_KEY       TYPE LVC_S_FCAT-KEY
                                 PV_COLPOS    TYPE LVC_S_FCAT-COL_POS
                                 PV_COLTEXT   TYPE LVC_S_FCAT-COLTEXT.

  CLEAR: GS_FCAT.

  GS_FCAT-FIELDNAME = PV_FIELDNAME.
  GS_FCAT-KEY       = PV_KEY.
  GS_FCAT-COL_POS   = PV_COLPOS.
  GS_FCAT-COLTEXT   = PV_COLTEXT.

  APPEND GS_FCAT TO GT_FCAT.

ENDFORM.
*&---------------------------------------------------------------------*
*& Form set_fieldcatlog
*&---------------------------------------------------------------------*
*& text
*&---------------------------------------------------------------------*
*&      --> P_
*&      --> P_
*&---------------------------------------------------------------------*
FORM SET_FIELDCATLOG USING PV_FIELD TYPE ANY
                               PV_VALUE TYPE ANY.

  DATA: LV_STR TYPE STRING.

  FIELD-SYMBOLS: <FS> TYPE ANY.

  LV_STR = 'GS_FCAT-' && PV_FIELD.
  ASSIGN (LV_STR) TO <FS>.

  IF <FS> IS ASSIGNED.
    <FS> = PV_VALUE.
  ENDIF.

  IF PV_FIELD = 'END'.
    APPEND GS_FCAT TO GT_FCAT.
    CLEAR: GS_FCAT.
  ENDIF.

ENDFORM.
